## Введение

OzonProfit - это приложение для аналитики и управления магазином на платформе Ozon. Приложение предоставляет полный набор инструментов для отслеживания продаж, анализа данных, управления финансами и администрирования товаров.

## Первые шаги

### Вход в систему
1. Откройте приложение в веб-браузере на `http://localhost:3000`
2. Введите ваши учетные данные Ozon API
3. Нажмите кнопку "Вход"

### Интеграция с API Ozon

Для полной работоспособности приложения необходимо:

1. Получить API токен в кабинете продавца Ozon
2. Указать Client ID и API Key в настройках приложения
3. Проверить подключение к API

## Дашборд

Дашборд - главная страница приложения, отображающая ключевые метрики вашего магазина.

### Что вы видите на дашборде:

- **Общие продажи**: Сумма всех продаж за текущий период
- **Количество заказов**: Общее количество поступивших заказов
- **Средний чек**: Среднее значение стоимости одного заказа
- **Рейтинг магазина**: Ваш текущий рейтинг на платформе Ozon
- **Последние заказы**: Таблица с информацией о последних 10 заказах
- **Графики**: Визуализация тенденций продаж по дням

### Как использовать дашборд:

1. Выберите период времени (Сегодня, Неделя, Месяц, Год) в верхней части экрана
2. Все метрики автоматически обновятся в соответствии с выбранным периодом
3. Нажимая на названия метрик, вы можете перейти к более подробному анализу
4. Используйте кнопку обновления для загрузки последних данных

## Аналитика

Раздел Аналитика предоставляет детальный анализ вашей деятельности на Ozon.

### Доступные отчеты:

#### Анализ товаров
- Список всех ваших товаров с показателями
- Информация о просмотрах, добавлениях в корзину и конверсиях
- Данные о возвратах и жалобах
- Сортировка по рейтингу, цене, популярности

#### Анализ по категориям
- Распределение продаж по категориям товаров
- Вклад каждой категории в общий доход
- Диаграммы и графики тенденций

#### Анализ трафика
- Источники трафика (поиск, рекомендации, прямые ссылки)
- География покупателей
- Поведение пользователей на вашем товаре

### Как анализировать данные:

1. Перейдите в раздел "Аналитика"
2. Выберите интересующий вас отчет
3. Установите нужные фильтры (дата, категория, товар)
4. Экспортируйте данные в CSV/Excel при необходимости
5. Используйте графики для визуализации тенденций

## Финансы

Раздел Финансы показывает полную картину вашего финансового состояния на платформе.

### Финансовые отчеты включают:

- **Выручка**: Общая сумма из продаж
- **Комиссии**: Размеры комиссий Ozon и платежных систем
- **Расходы**: Расходы на логистику, хранение и другие услуги
- **Прибыль**: Чистая прибыль (Выручка - Комиссии - Расходы)
- **История платежей**: Все переводы средств на ваш счет

### Управление финансами:

1. Откройте раздел "Финансы"
2. Выберите период анализа
3. Просмотрите детали каждой статьи расходов
4. Проанализируйте динамику прибыльности
5. Экспортируйте отчеты для бухгалтерии

## Товары

Раздел Товары предоставляет полный контроль над вашим каталогом.

### Управление товарами:

#### Просмотр списка товаров
1. Откройте раздел "Товары"
2. Используйте фильтры для поиска товара
3. Сортируйте по названию, цене, количеству, рейтингу

#### Редактирование товара
1. Нажмите на товар в списке
2. Отредактируйте нужные параметры:
   - Название и описание
   - Цена и остаток
   - Характеристики
   - Изображения
3. Нажмите "Сохранить изменения"

#### Синхронизация остатков
1. В списке товаров найдите кнопку "Синхронизировать остатки"
2. Приложение сверит ваши остатки с данными Ozon
3. Конфликты будут показаны отдельно

## Настройка API

### Получение API токена:

1. Перейдите в личный кабинет Ozon Seller
2. Откройте раздел "Программирование" > "API"
3. Создайте новый API ключ
4. Сохраните Client ID и API Key

### Добавление API ключа в приложение:

1. В приложении откройте "Настройки"
2. Перейдите на вкладку "API"
3. Введите Client ID
4. Введите API Key
5. Нажмите "Проверить подключение"
6. После успешной проверки нажмите "Сохранить"

### Тестирование подключения:

```bash
curl -X GET "http://localhost:8000/api/test" \
  -H "Content-Type: application/json"
```

Если подключение активно, вы получите ответ со статусом 200 OK.

## Типичные сценарии использования

### Сценарий 1: Ежедневный мониторинг продаж

1. Откройте приложение каждое утро
2. Проверьте Дашборд для быстрого обзора ночных продаж
3. Посмотрите новые заказы в разделе "Последние заказы"
4. Проверьте поступившие отзывы и вопросы
5. При необходимости отредактируйте цены товаров

### Сценарий 2: Анализ популярных товаров

1. Перейдите в Аналитика > Анализ товаров
2. Отсортируйте товары по числу просмотров
3. Определите топ-5 популярных товаров
4. Проанализируйте конверсию этих товаров
5. Убедитесь, что достаточно остатков популярных товаров
6. Рассмотрите возможность расширения ассортимента похожих товаров

### Сценарий 3: Финансовый анализ

1. Откройте раздел Финансы
2. Выберите месячный период анализа
3. Просмотрите полные расходы и комиссии
4. Рассчитайте маржу прибыли
5. Выявите товары с низкой рентабельностью
6. Скорректируйте цены или снимите товары с продажи

## Решение проблем

### Проблема: Данные не обновляются

**Решение:**
1. Проверьте подключение к интернету
2. Нажмите кнопку "Обновить" в приложении
3. Проверьте, активен ли API токен в кабинете Ozon
4. Перезагрузите страницу браузера (Ctrl+R)
5. Очистите кэш браузера (Ctrl+Shift+Delete)

### Проблема: Ошибка подключения к API

**Решение:**
1. Перейдите в Настройки > API
2. Проверьте правильность введения Client ID и API Key
3. Убедитесь, что токен не истек (создайте новый при необходимости)
4. Нажмите "Проверить подключение"
5. Если ошибка повторяется, обратитесь в техподдержку Ozon

### Проблема: Медленная работа приложения

**Решение:**
1. Закройте лишние вкладки браузера
2. Очистите кэш браузера
3. Обновите браузер до последней версии
4. Убедитесь, что скорость интернета достаточна (минимум 2 Мбит/с)
5. Перезагрузите приложение

### Проблема: Некоторые товары не отображаются

**Решение:**
1. Проверьте, что товары активны на платформе Ozon
2. Убедитесь, что у товаров установлены правильные статусы
3. Попробуйте синхронизировать каталог
4. Проверьте фильтры в разделе Товары
5. Обновите страницу
